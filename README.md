## Getting Started

### Dependencies

* Docker compose (and internet connection)

### Installing

* somewhere on github

### Executing program

* Clone the project folder from github
* navigate to the project folder
* run docker compose to run the app

```
cd path/to/my/vejr_app

docker compose build
docker compose up
```
